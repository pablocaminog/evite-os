# AI Image Prompts for Party Invitations

Generate custom cover images with ChatGPT, Gemini, Midjourney, or any image-generation tool.
Upload the result on the Create Party page (JPEG, PNG, or WebP, max 5MB).

## Tips

- Target **1200×630px** (landscape) for best display on the invitation page.
- Ask for "flat illustration" or "digital art" style for clean, kid-friendly results.
- Add "bright colors, white background" for designs that look great over the hero gradient.

## Using ChatGPT

1. Go to [chatgpt.com](https://chatgpt.com) and start a new chat
2. Type: **"Generate an image:"** followed by your chosen prompt above
3. Right-click the image → Save As
4. Upload on the Create Party page

## Using Gemini

1. Go to [gemini.google.com](https://gemini.google.com)
2. Click the image icon in the prompt bar
3. Paste your prompt and generate
4. Download and upload on the Create Party page

## Using Midjourney

1. Use the prompt directly in Discord with `/imagine`
2. Add `--ar 2:1` for landscape
3. Download and upload on the Create Party page
